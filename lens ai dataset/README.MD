# 🔬 LensAI Dataset Management Platform

A powerful platform for managing, organizing, and processing AI/ML datasets efficiently.

## 🚀 Features
- 📂 Dataset upload & management
- 🏷️ Smart labeling & annotation
- 📊 Dataset statistics & insights
- 🔍 Search & filter datasets
- 📤 Export in multiple formats
